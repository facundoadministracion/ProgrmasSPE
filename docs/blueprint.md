# **App Name**: LR Manager

## Core Features:

- Dashboard Overview: Display a summary of key metrics, including total participants, alerts, and registered payments.
- Participant Management: Enable adding, viewing, and editing participant data, including personal information, program details, and contact information.
- Attendance Tracking: Provide a module to record and manage participant attendance, with departmental breakdowns.
- Payment Upload: Offer a wizard-like interface for bulk uploading of payment data via CSV files, with data validation and error handling.
- Program Analytics: Provide a section to analyze programs statistics for team performance analysis.
- Alert Generation: Flag participants who reach their age limits. Use AI tool for flagging when the payments limits are reached.

## Style Guidelines:

- Primary color: Dark slate blue (#475569) for a professional and trustworthy feel.
- Background color: Light gray (#F9FAFB) for a clean and neutral backdrop.
- Accent color: Teal green (#0D9488) to highlight key actions and elements.
- Body and headline font: 'Inter', a sans-serif font providing a clean, modern and neutral aesthetic
- Use 'lucide-react' icons throughout the application for a consistent and modern look.
- Employ a clean and organized layout with clear navigation and intuitive user flows.
- Subtle animations, such as fade-in effects and smooth transitions, to enhance the user experience.
